# orien-research — 下見支援アプリ（サービス2）

オリエンテーリング地図作成者が現地下見を効率的に行うためのスマホPWA。
電波なし環境での利用を前提とする。

## 機能概要

| フェーズ | 機能 |
|---------|------|
| 事前準備（電波あり） | サービス1のGeoJSONを読み込み、地図をキャッシュ |
| 現地作業（オフライン） | CP設置・調査メモ記録（ポイント/ライン/エリア）・写真撮影 |
| 出力 | ZIPファイル（survey.geojson + photos/） |

## 技術スタック

- React 18 + TypeScript + Vite
- MapLibre GL JS（地図表示）
- 国土地理院タイル（PWAキャッシュ・Service Worker）
- IndexedDB（idb）— データ永続化・自動保存
- JSZip — ZIP出力
- vite-plugin-pwa — PWA対応

## セットアップ

```bash
npm install
npm run dev        # 開発サーバー（http://localhost:5173）
npm run build      # プロダクションビルド
npm run preview    # ビルド結果のプレビュー
```

## 使い方

1. **事前準備画面**: サービス1で出力したGeoJSONを読み込み、オフライン保存範囲の地図をキャッシュ
2. **「現地作業を開始」**: 地図画面へ遷移
3. **CP設置**: 「現在地へCP設置」ボタンでGPS位置を取得してCP情報を入力
4. **調査メモ**: ポイント/ライン/エリアを地図上にタップで描画してメモを記録
5. **エクスポート**: ☰メニューから「📦 ZIPでエクスポート」でサービス3向けのZIPをダウンロード

## 地図上の操作

### 設置CP（赤丸）

| 操作 | 動作 |
|------|------|
| シングルタップ | 選択（黄色リング＋ドラッグハンドル表示） |
| ドラッグ | 位置をリアルタイム移動 |
| ダブルタップ | CP編集画面を開く |
| 空きエリアをタップ | 選択解除 |

### 調査メモ（ポイント / ライン / エリア）

| 操作 | 動作 |
|------|------|
| シングルタップ | 選択（黄色ハイライト＋頂点ハンドル表示） |
| 頂点ハンドルをドラッグ | 頂点を移動 |
| ライン/エリア本体をドラッグ | 全体を移動 |
| ダブルタップ | 編集画面を開く |
| 空きエリアをタップ | 選択解除 |

### 拡大鏡

CP・ポイント・ライン/エリア頂点のドラッグ中、ドラッグ点の上方に拡大鏡（ズームレベル18）を表示。精密な位置合わせに使用。

## 出力フォーマット

```
{エリア名}_{日付}_s2_v1.zip
  ├── survey.geojson
  └── photos/
        ├── cp_001_01.jpg
        └── sm_001_01.jpg
```

## GPS取得失敗時

1. 再試行
2. 緯度経度を直接入力
3. 地図上で位置を指定

## Undo

- スマホ: 画面上の「↩ Undo」ボタン
- PC: Ctrl+Z
- 最大50ステップ
